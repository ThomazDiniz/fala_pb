## F5-TTS — treino do zero (`filtered_zero_experiment`)

Treino completo a partir de **pesos aleatorios** (sem checkpoint FirstPixel), mesmo corpus usado nos finetunes anteriores (~41k amostras, `filtered_char`).

| Item | Valor |
|---|---|
| Checkpoint final | `model_last.pt` (epoca **150**) |
| Learning rate | **7,5×10⁻⁵** |
| Batch | 1000 frames / GPU, max 64 amostras |
| Tempo total | **~244 h** (~10 dias de parede) |
| Melhor eval loss | **0,590** (epoca 136) |
| Eval loss final | **0,644** (epoca 150) |

Referencia de sintese: `PB_0991.wav` (copia em [`f5tts_do_zero/PB_0991_ref.wav`](f5tts_do_zero/PB_0991_ref.wav)); texto de prompt de voz: *"Fabiana hesitou, resmungou como fazia sempre que ele dirigia uma palavra."*

### Resultado

**Negativo.** Apesar da loss e do eval loss convergirem no log, as **cinco sinteses do checkpoint final nao produzem fala audivel** — output incompreensivel / nao utilizavel na escuta. **Nao recomendavel** seguir para avaliacao automatica nem para [validacao perceptual](/fala_pb/18/#experimento-de-validacao-perceptual) com humanos.
