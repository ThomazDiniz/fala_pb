# Experimento 14 — curadoria de dados, ruído e próximos treinos

Plano inicial deste experimento (checklist herdada das próximas atividades do 13):

- [x] Filtrar massa de dados.
- [x] Quando filtrar, mostrar exemplos de entrevistas que têm ruído e remover ruído.
- [ ] Experimento com mais dados e menos épocas (aumentar o volume de dados tende a permitir reduzir épocas; avaliar reduzir ainda mais a quantidade de épocas).
- **Risco:** verificar erros de ajuste fino.
- [ ] Unsloth: verificar questão da acentuação para sintetizar.
- [ ] Finetune do Unsloth.
- [ ] Treinamento mais robusto por alguns dias do modelo **F5-TTS** para verificar se obtemos algum ganho positivo com isso.
